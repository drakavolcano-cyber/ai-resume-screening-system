# 🤖 AI Resume Screening System

An AI-powered HR automation workflow built with **n8n**, **OpenAI**, and **Google Workspace**.

---

## 📌 Overview

This project automates the resume screening process using AI. It helps recruiters review applications faster by analyzing resumes, checking duplicate applicants, scoring candidates, and storing results automatically.

---

## ✨ Features

- 📄 AI Resume Analysis
- ⭐ Candidate Scoring
- 🔍 Duplicate Email Detection
- 📧 Automatic Email Notification
- 📊 Google Sheets Integration
- ⚡ Automated Workflow using n8n

---

## 🛠 Tech Stack

- n8n
- OpenAI API
- Google Sheets
- Gmail
- JavaScript
